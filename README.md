# EmailBomb
This project is for continued discussion only and must not be used for illegal activities.  This is an email bombing program.
Language: Java
Environment: JDK 1.8+
## How to use the EmailBomb?
First you need to download Java at [Java Download](https://www.java.com/en/download/manual.jsp),And You need to download this project.
Then open the Shell
```bash
java -jar EmailBombV1-1.0.jar
```
Then follow the prompts

## How to turn on the app password?
As is well known, most email services do not allow direct password login to this program.
So we need an App Password
How to enable Gmail App Password?
you need [login your Google Account](https://myaccount.google.com/) Security & sign-in --> Turn on Two-step Verification.
And [Set an App Password](https://myaccount.google.com/apppasswords) to set an app password
